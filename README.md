# cafe-sales-practice
Taming messy café sales data ☕📊: A practice project in cleaning, analyzing, and visualizing insights.

Disclaimer: This is a practice project to sharpen my data cleaning, visualization, and analytical skills. I had some AI guidance along the way, but don’t worry, I didn’t let it do the thinking for me. All decisions, reasoning, and insights are my own.

**Project Overview**
Objective:
  Clean a messy café sales dataset
  Explore trends in sales, items, payment methods, and locations
  Visualize insights using Power BI (and sprinkle a little Python magic)
  Practice thinking like a data analyst, not just a number-cruncher

Why I chose this dataset:
  This is the first dataset I ever cleaned! I love cafés, but my social anxiety makes it awkward to visit one without overthinking that i'm being performative. So I thought, why not just analyze café sales data from the comfort of my own desk?
